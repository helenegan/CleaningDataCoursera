tidy_data.txt contains a data set generated from data collected from the accelerometers from the Samsung Galaxy S smartphone.
run_analysis.R script was used to generate this data set.
The script performs the following transformations  to obtain this tidy data set:
* it merges the training and the test sets to create one data set.
* it extracts only the measurements on the mean and standard deviation for each measurement.
* it uses descriptive activity names from activity_labeles.txt to name the activities in the data set
* it appropriately labels the data set with descriptive variable names from features.txt.
* it creates a second, independent tidy data set with the average of each variable for each activity and each subject.

You may find this data set in tidy_data.txt in this repo.

Please refer to features_info.txt in https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip for datailed description of the following variables.
